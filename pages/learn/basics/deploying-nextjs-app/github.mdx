Before we deploy, let’s push our Next.js app to [GitHub](https://github.com/zeit/next.js) if you haven’t done so already. This will make deployment easier.

- On your personal GitHub account, create a new repository called `nextjs-blog`.
- The repository can be public or private. You do **not** need to initialize it with a README or other files.
- Take a look at [the help page on GitHub](https://help.github.com/en/github/getting-started-with-github/create-a-repo) if necessary.

Then, push the Next.js app we’ve created to this repository. To push to GitHub, you can use [GitHub Desktop](https://desktop.github.com/) or run the following commands (replace `<username>` with your GitHub username):

```bash
git remote add origin https://github.com/<username>/nextjs-blog.git
git push -u origin master
```

Once your GitHub repository is ready, continue to the next page.
